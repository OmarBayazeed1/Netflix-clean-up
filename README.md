## Project Overview

This project is part of the data analysis learning path on [roadmap.sh](https://roadmap.sh). Its primary goal is to perform data cleaning on the Netflix Titles dataset obtained from Kaggle. The notebook demonstrates how to transform raw, messy data into a tidy, analysis-ready format.

- **The Problem:** The raw dataset contains missing values, mixed-type columns (e.g., duration as "90 min"), and unparsed date columns.
- **The Solution:** This project uses `pandas` to systematically inspect, clean, and standardize the data for further analysis.
